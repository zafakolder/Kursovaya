# 🛍️ Сувенирный магазин

**Интернет-магазин сувенирной продукции** с разделением на бэкенд (Django REST API) и фронтенд (React SPA).  
Проект выполнен по **траектории Б** в рамках курсовой работы по дисциплине «Технология разработки программного обеспечения».

---

## 📖 О проекте

«Сувенирный магазин» – это веб-приложение, которое позволяет:

- 🔍 **Просматривать** каталог товаров с поиском, фильтрацией по категориям и пагинацией.
- 🛒 **Добавлять** товары в корзину, изменять количество и оформлять заказы.
- 👤 **Регистрироваться** и входить в систему с помощью JWT-токенов.
- 🛠️ **Администраторам** – управлять товарами и категориями через админ-панель Django и интерфейс сайта.

Проект разработан по **траектории Б** (Django REST + React SPA + JWT).

## 📋 Основной функционал

### Публичная часть (доступна всем)
- Просмотр списка товаров (пагинация по 10 штук)
- Поиск по названию и описанию
- Фильтрация по категориям
- Детальная страница товара

### Приватная часть (для авторизованных пользователей)
| Роль | Возможности |
|------|-------------|
| **Пользователь** | ➕ Добавление товаров в корзину<br>➖ Изменение количества товаров<br>📦 Оформление заказа с указанием адреса<br>📜 Просмотр истории заказов |
| **Администратор** (`is_staff`) | ➕ Создание новых товаров<br>✏️ Редактирование и удаление товаров<br>🗂️ Управление категориями через админку<br>👤 Управление пользователями |
