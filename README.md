Сувенирный магазин — интернет-магазин сувенирной продукции
Сувенирный магазин — это веб-приложение для продажи сувениров. Пользователи могут просматривать товары, добавлять их в корзину и оформлять заказы. Администраторы управляют каталогом товаров и категориями.

Проект разработан в рамках курсовой работы по дисциплине «Технология разработки программного обеспечения» (траектория Б: Django REST + React SPA + JWT).

📋 Основной функционал
Публичная часть (доступна всем)
Просмотр списка товаров (с пагинацией)

Фильтрация товаров по категориям

Поиск товаров по названию или описанию

Регистрация и аутентификация (JWT)

Просмотр детальной страницы товара

Приватная часть (для авторизованных пользователей)
Роль	Возможности
Пользователь	Добавление товаров в корзину, оформление заказа, просмотр своих заказов
Администратор (is_staff)	Создание, редактирование и удаление товаров через интерфейс (и через админ-панель Django)
Дополнительный функционал
Корзина — пользователи могут добавлять товары, изменять количество, очищать корзину

Оформление заказа — при оформлении корзина превращается в заказ с фиксацией цены

Пагинация — на главной странице выводится по 10 товаров

Фильтрация и поиск — поиск по названию и описанию, фильтр по категории

⚙️ Установка и запуск
1. Клонируйте репозиторий
bash
git clone <url-репозитория>
cd souvenir_shop
2. Настройка бэкенда
bash
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser   # для входа в админку
python manage.py runserver
Сервер будет доступен по адресу http://127.0.0.1:8000.

3. Настройка фронтенда
Откройте новый терминал:

bash
cd frontend
npm install
npm start
Фронтенд будет доступен по адресу http://localhost:3000.

🔗 API Endpoints
Метод	Эндпоинт	Назначение	Доступ
GET	/api/products/	Список товаров (с пагинацией, фильтрацией, поиском)	Все
POST	/api/products/	Создание товара	Админ
GET	/api/products/{id}/	Детальный просмотр товара	Все
PUT/PATCH	/api/products/{id}/	Обновление товара	Админ
DELETE	/api/products/{id}/	Удаление товара	Админ
GET	/api/categories/	Список категорий	Все
GET	/api/cart/	Получить корзину	Авторизованный
POST	/api/cart/add/	Добавить товар в корзину	Авторизованный
POST	/api/cart/remove/	Уменьшить количество или удалить товар	Авторизованный
DELETE	/api/cart/clear/	Очистить корзину	Авторизованный
POST	/api/cart/checkout/	Оформить заказ	Авторизованный
GET	/api/orders/	Список заказов пользователя	Авторизованный
POST	/api/auth/register/	Регистрация	Все
POST	/api/auth/login/	Получение JWT-токенов	Все
POST	/api/auth/token/refresh/	Обновление access-токена	Все
GET/PUT	/api/auth/profile/	Получение/обновление профиля	Авторизованный
📁 Данные для входа в админ-панель
После создания суперпользователя (createsuperuser) вы можете войти в админку Django по адресу http://127.0.0.1:8000/admin/ и управлять товарами, категориями, заказами и пользователями.
